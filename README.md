# برنامج جرد الأجهزة - Inventory Management App

تطبيق موبايل متقدم لإدارة وجرد الأجهزة والمعدات، مبني بـ React Native و Expo.

## المميزات الرئيسية

### 📱 إدارة الأجهزة
- ✅ إضافة وتعديل وحذف الأجهزة
- ✅ تصنيف الأجهزة (النوع، الحالة، الموقع)
- ✅ البحث والفرز المتقدم
- ✅ **رموز QR للأجهزة** 🔲

### 🔲 رموز QR
- ✅ توليد رمز QR فريد لكل جهاز
- ✅ عرض معلومات الجهاز في الرمز
- ✅ سهولة المسح الضوئي للبيانات
- ✅ استخراج البيانات بسرعة

### 🔄 تتبع الحالة
- ✅ حالات متعددة: نشط 🟢، معطل 🔴، صيانة 🟡، مستبعد ⚫
- ✅ تغيير الحالة بسهولة من واجهة التطبيق
- ✅ تاريخ التعديلات التلقائي
- ✅ ملاحظات وتعليقات

### 📊 الصيانة والتقارير
- ✅ سجل الصيانة التاريخي
- ✅ معلومات الفني والتكاليف
- ✅ تقارير إحصائية

### 🎨 واجهة سهلة الاستخدام
- ✅ تصميم عصري وبديهي
- ✅ دعم اللغة العربية كامل
- ✅ رموز تعبيرية وألوان واضحة
- ✅ ملمس Native

## المتطلبات

- Node.js 16+
- npm أو yarn
- Expo CLI: `npm install -g expo-cli`
- Android Studio (لـ Android) أو Xcode (لـ iOS)

## التثبيت والتشغيل

```bash
# استنساخ المشروع
git clone https://github.com/fawazalotaibi40-beep/kamcj.med.sa.git
cd kamcj.med.sa

# تثبيت الاعتماديات
npm install

# تشغيل التطبيق
npm start          # التطبيق على الجوال (Expo Go)
npm run ios        # iOS
npm run android    # Android
npm run web        # على المتصفح
```

## كيفية الاستخدام

### 1. إضافة جهاز جديد
```
1. اضغط على زر (+) الإضافة
2. أدخل معلومات الجهاز
3. احفظ الجهاز
```

### 2. عرض رمز QR
```
1. اختر الجهاز من القائمة
2. اضغط على "عرض رمز QR"
3. يظهر الرمز بصيغة QR قابلة للمسح
```

### 3. تغيير حالة الجهاز
```
1. فتح تفاصيل الجهاز
2. اختر الحالة الجديدة
3. سيتم التحديث تلقائياً
```
