# برنامج جرد الأجهزة - Inventory Management App

تطبيق موبايل متقدم لإدارة وجرد الأجهزة والمعدات، مبني بـ React Native و Expo.

## المميزات الرئيسية

إضافة وتعديل وحذف الأجهزة
تصنيف الأجهزة بالنوع والحالة والموقع
البحث والفرز المتقدم
رمز QR للأجهزة
حالات متعددة: نشط، معطل، صيانة، مستبعد
تاريخ الحالة والتعديلات
ملاحظات وتعليقات
جدول صيانة دوري
سجل الصيانة التاريخي
تقارير مفصلة
قاعدة بيانات محلية (SQLite)
تصدير/استيراد البيانات
نسخ احتياطية تلقائية

## المتطلبات

- Node.js 16+
- npm أو yarn
- Expo CLI
- Android Studio أو Xcode

## التثبيت والتشغيل

npm install
npm start
